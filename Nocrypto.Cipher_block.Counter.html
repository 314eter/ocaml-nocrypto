Module Nocrypto.Cipher_block.Counter

module Counter: sig .. end
BE counter function.

Each incrX cs i increments X-sized block of cs at the offset i by one, returning true if an overfow occurred (and the block is now zeroed-out).

Each addX cs i n adds n to the X-sized block.
